Images

A image is a picture included in your web page.

To create an image, use the <img> tag. The <img> tag is empty and only contains attributes. It does not have a closing tag.

Example: Red apple

The <img> tag has two required attributes:

  • src - the path to the image
  • alt - the alt text to the image

Image Maps

A image map is an image with clickable areas.

Lists

A list is a group of related items that are displayed together. Lists provide developers and users an easy way to structure and read these items.

HTML provides three types of lists:

  • Unordered Lists
  • Ordered Lists
  • Definition Lists

Unordered Lists

An unordered list is a list of related items that do not have any order. An example of an unordered list is a grocery list. You do not need to purchase your groceries in any particular order, you just need to purchase that item on the list.

To create an unordered list, use the <ul> element. To add an item to the unordered list, use the <li> element.

Unordered List Example - Grocery List

  • Eggs
  • Bread
  • Hot Sauce

Ordered Lists

An ordered list is a list of related items that appear in a certain order. An example of an ordered list is a recipe. You need to follow each step in order to make the recipe right.

To create an ordered list, use the <ol> element. To add an item to the unordered list, use the <li> element.

Ordered List Example - Cooked Egg Recipe

  1. Crack the egg.
  2. Add egg to the pan.
  3. Cook egg until done.
